# Hitesh Shimpi Portfolio

Personal portfolio website built with React and deployed on Vercel.

## Tech Stack

- React
- React Bootstrap
- CSS
- Vercel

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm start
```

3. Build for production:

```bash
npm run build
```

## Project Structure

- `src/components/` contains the UI sections and shared components
- `src/Assets/` contains static assets like images and the resume PDF
- `public/` contains the HTML template and public files

## Deployment

This project can be deployed to Vercel as a static React application.

## Notes

- Routing is handled with React Router
- `vercel.json` includes the SPA rewrite needed for direct route refreshes
